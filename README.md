# Investor Platform Backend

Este proyecto contiene el backend de la plataforma de inversores.

## Requisitos

- Docker

## Cómo ejecutar la aplicación

1. Clona este repositorio.
2. Asegúrate de tener Docker instalado y en ejecución.
3. En la raíz del proyecto, ejecuta el siguiente comando para construir y levantar el contenedor de la aplicación:

   ```bash
   docker-compose up -d
   ```

La aplicación estará disponible en `http://localhost:8080`.

Para detener la aplicación, ejecuta:

```bash
docker-compose down
```
